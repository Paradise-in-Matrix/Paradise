
** Core

The initial namespace in which we start the application.
We'll aim to keep everything neatly segmented, but as we start scaffolding
components it is likely that there will be some residual items until we finalize
pieces.

**** Reframe event and subscriptions 

We intend to this to be passed as a callback where needed and handle any client
operations here up until the point in which the client is 'started'.
#+begin_src clojurescript :tangle ../../src/client/core.cljs
(re-frame/reg-event-db
 :sdk/set-client
 (fn [db [_ raw-client]]
   (assoc db :client raw-client)))

(re-frame/reg-sub
 :sdk/client
 (fn [db _]
   (:client db)))

#+end_src

**** Initialize the application!

Provides the entry-point for the application that Vite and the index.html enter from.
#+begin_src clojurescript  :tangle ../../src/client/core.cljs
(defn ^:export init []
  (app/init)
  (logger/init!)
  (log/debug  "Entering Paradise!")
  )
#+end_src
